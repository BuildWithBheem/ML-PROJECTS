## User Manual

Enter the following details:

| Input Field                      | Description                         |
| -------------------------------- | ----------------------------------- |
| Study Hours Per Week             | Number of hours studied weekly      |
| Attendance Percentage            | Overall attendance percentage       |
| Assignments Submitted Percentage | Percentage of assignments submitted |
| Sleep Hours Per Day              | Average daily sleep hours           |
| Coding Practice Hours Per Week   | Weekly coding practice hours        |
| Previous Semester Backlogs       | Number of existing backlogs         |
| Social Media Usage Hours Per Day | Average daily social media usage    |
| Semester Number                  | Current semester                    |

After entering all values, click:

```text
Predict CGPA
```

The application will return the predicted CGPA based on the trained Polynomial Regression model.
